# 🎬 CineMatch – Movie Recommendation Web App

CineMatch is a dynamic web application that helps users discover movies based on their preferences. It provides personalized suggestions, highlights underrated films, and shows where movies are available to watch.

---
# 🎬 Cinematch for Cinephiles

Discover movies tailored for true cinephiles.

## 🌐 Live Demo
👉 https://cinematch-for-cinephiles.netlify.app/

## ✨ Features
- 🎭 Genre & language-based filtering  
- 💎 Hidden gems recommendation  
- 📺 OTT platform availability (Netflix, Prime, etc.)  
- 🎥 Trailer popup inside app  
- ❤️ Favorites system (localStorage)  
- 📱 Responsive design  

---

## 🛠 Tech Stack
- HTML  
- CSS  
- JavaScript  
- TMDB API
